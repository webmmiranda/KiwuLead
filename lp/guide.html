Guía de Uso KiwüLead

Bienvenido a la documentación oficial de KiwüLead. Esta guía te llevará paso a paso desde la configuración inicial hasta la automatización completa de tu proceso de ventas.

1
Primeros Pasos

Instalar y crear usuario Soporte

El primer usuario creado en el sistema tendrá rol de Administrador Global (Soporte). Este usuario tiene acceso a todas las configuraciones críticas del sistema.

  1. Accede a la URL de instalación proporcionada por tu proveedor.
  2. Completa el formulario de registro inicial.
  3. Verifica que el rol asignado sea "Admin/Soporte".

Configurar Email de Sistema

Para que KiwüLead envíe notificaciones (recuperación de contraseña, alertas de nuevos leads), debes configurar el servidor SMTP.

Ruta de acceso:

Configuración > Email > Configuración SMTP
  • Ingresa el Host SMTP (ej: smtp.gmail.com).
  • Puerto (ej: 587 para TLS o 465 para SSL).
  • Usuario y Contraseña del correo.
  • Haz clic en "Probar Conexión" para verificar.

2
Configuración Global de Empresa

Personaliza la apariencia y los datos fiscales de tu instancia de KiwüLead. Estos datos aparecerán en las facturas y correos generados.

Datos Generales

  • Nombre de Empresa: Visible en el título del navegador.
  • Logo Cuadrado: Sube tu logo en formato PNG/JPG (recomendado 512x512px).
  • Sitio Web: URL principal de tu negocio.

Datos Fiscales

  • Industria: Categoría de tu negocio.
  • Moneda Base: Moneda principal para reportes (USD, MXN, CLP, etc.).
  • ID y Dirección Física: RUT/NIT y dirección para pie de página.

3
Gestión de Usuarios

Crear Gerentes y Vendedores

KiwüLead permite crear usuarios ilimitados. Es importante asignar el rol correcto para mantener la seguridad de los datos.

Gerente
Rol de Gerente

Puede ver todos los leads, asignar contactos, ver reportes globales y gestionar configuraciones básicas.

Vendedor
Rol de Vendedor

Solo puede ver y gestionar los leads que se le han asignado. No tiene acceso a configuraciones globales.

Pasos para crear un usuario:

  1. Ve a Configuración > Equipo.
  2. Haz clic en "Invitar Miembro".
  3. Ingresa el correo electrónico y selecciona el Rol.
  4. El usuario recibirá un correo para configurar su contraseña.

5
Cómo Obtener Leads

Conexión vía Webhook (Formularios Web)

Puedes conectar cualquier formulario (Elementor, Contact Form 7, Typeform) enviando los datos a la URL de Webhook única de tu cuenta.

POST https://tu-crm.com/api/webhook.php?api_key=TU_API_KEY

Asegúrate de enviar los campos JSON con los nombres: name, email, phone.

Meta Ads y WhatsApp Business

Meta Ads (Facebook/Instagram)

Utiliza la integración nativa para que los leads de tus formularios instantáneos (Lead Ads) aparezcan automáticamente en el Pipeline "Nuevos Leads".

WhatsApp Business

Escanea el código QR en Configuración > WhatsApp para vincular tu dispositivo. Todos los mensajes entrantes de números desconocidos pueden crear un lead automáticamente si activas la opción.
